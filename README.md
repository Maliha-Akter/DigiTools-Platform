#  DigiTools Platform

## 📌 Description
DigiTools Platform is a modern web application that allows users to explore premium digital tools, add them to a cart, and experience a smooth and interactive interface. The project focuses on simplicity, performance, and user-friendly design.

---

## 🛠️ Technologies Used
- ⚛️ React.js  
- 🎨 Tailwind CSS & DaisyUI  
- 📜 JavaScript (ES6)  
- 🌐 HTML5 & CSS3  

---

## ✨ Key Features
- 🛒 Product can be added to Cart & Removed from Cart functionality  
- 🔄 Dynamic button state (Buy Now → Added to Cart)  
- 📱 Fully responsive design for all devices
